# TDMPoll

Mini web-app de vote (100% statique) pour choisir un nom DEV et un nom PROD.

## Déploiement GitHub Pages (très court)

1. Poussez ces fichiers sur la branche `main`.
2. Dans GitHub → **Settings** → **Pages** → Source : `Deploy from a branch` → `main` → `/root`.
3. Sauvegardez : l’app sera disponible via l’URL GitHub Pages fournie.
